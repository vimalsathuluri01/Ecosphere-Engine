## 🚀 System Deployment (Technical Instructions)

### Prerequisites
- **Runtime**: Node.js 18.x or higher (Bun recommended for performance)
- **Database**: SQLite (Default)
- **Framework**: Next.js 15 (App Router)

### 1. Environment Configuration
Create a `.env` file in the root directory:
```env
DATABASE_URL="file:./dev.db"
# NEXTAUTH_SECRET="your-secret-here" (Optional for sign-in)
```

### 2. Dependency Installation
Execute with clinical precision:
```bash
npm install
# or if using bun
bun install
```

### 3. Database Initialization
Prepare the persistence layer for multi-dimensional entity data:
```bash
npx prisma generate
npx prisma db push
```

### 4. Launching the Diagnostic Engine
Start the local development environment:
```bash
npm run dev
```
The command center will be accessible at [http://localhost:3000](http://localhost:3000).
